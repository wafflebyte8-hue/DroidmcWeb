Android Minecraft Server Panel

DroidMC

Run and manage a Minecraft server directly from your Android phone through Termux. Choose the slower stable track or the faster beta track, then install with one command.

Choose a Channel Watch Tutorials

What It Does

Server control, live console, downloads, backups, crash recovery, auth, HTTPS, and mobile-friendly management from a browser on the same WiFi.

Who It Is For

People who want a lightweight Minecraft server panel on Android using Termux without setting up a full VPS or desktop host.

How This Site Helps

Use this page as the hub for stable installs, beta installs, tutorial videos, and direct links to the GitHub repos.
